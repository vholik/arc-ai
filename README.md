# Arc AI

A gym progress chatbot with AI-powered photo analysis and personalized coaching.

## Features

- **Photo Analysis** - Track visual progress with standardized comparisons
- **Workout Logging** - Log exercises, sets, reps, RPE, and detect PRs
- **Adaptive Coaching** - Personalized training plans that adjust to your progress
- **Progress Tracking** - Weight, measurements, and trend analysis
- **Smart Insights** - Plateau detection, consistency scoring, weekly summaries

## Get Started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Open in [Expo Go](https://expo.dev/go), iOS simulator, or Android emulator

## Project Structure

```
app/           # Expo Router screens
components/    # Reusable UI components
  chat/        # Chat-specific components
  ui/          # Base UI components
constants/     # Theme and configuration
hooks/         # Custom React hooks
```
